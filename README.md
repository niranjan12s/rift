Resonant Interference Framewoek for Transformations(RIFT): 
This Github Repo is made to archive the initial code for the RIFT Architecture for generation and the baseline Code used for comparison. Google Colab was used during development , hence the code has been structured as .py files each pertaining to a cell for ease of use. The nature of the dataset used was small due to colab-based training restrictions in the free version. The paper aims to question the foundational assumptions of how meaning should be represented, constructed, and generated. And in doing so, it opens a rift in the standard landscape of generative modelling: a new path, built not on memorization or optimization, but on structure, resonance, and internal coherence.
